### Aim

Monitor the temperature and humidity around with DHT11.

---

### Target

eYFi-Mega dev board (ATmega2560) [https://www.e-yantra.org/products](https://www.e-yantra.org/products)

---

### Connections

- DHT11 Data   ->   PA3   ->   Arduino pin 25
